What the project does

This project improves recommendation quality by adding sentiment awareness to the recommendation process. Instead of relying only on plain matching or frequency logic, it uses user sentiment and text features to better understand what content feels more relevant and useful.

By integrating VADER and TF-IDF, the system becomes more context-aware and can rank suggestions with stronger relevance to user expression and feedback.
